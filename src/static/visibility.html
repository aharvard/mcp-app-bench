Registered Visibility Test Tools

Each tool below has a different visibility value. Only tools that include "model" should be sent to the LLM.

visibility-both ["model", "app"] — ✅ Model SHOULD see this
visibility-app-only ["app"] — 🚫 Model should NOT see this
visibility-model-only ["model"] — ✅ Model SHOULD see this
visibility-default (no visibility field) — ✅ Model SHOULD see this (default)

How to Test

  1. Ask: "List your visibility- tools"
  2. Expect: visibility-both, visibility-model-only, visibility-default
  3. Missing: visibility-app-only (hidden from model)
  4. Verify: asking the model to call visibility-app-only should fail
